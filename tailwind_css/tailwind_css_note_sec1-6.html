Tailwind CSS - border（枠線）のスタイル指定

🧱 基本構文

border-[方向]-[太さ] [色] [スタイル] [丸み]

✅ 1. 枠線の表示・非表示

クラス名説明
border全体に枠線（1px）
border-0枠線を非表示
border-2, border-4太さ指定
border-t, border-b上下左右ごとの枠線
<div class="border border-gray-400">全方向に枠線</div>
<div class="border-t-2 border-blue-500">上だけ太い青の枠線</div>

🎨 2. 枠線の色

<div class="border border-red-500">赤い枠線</div>
<div class="border-t border-green-300">上だけ薄緑色</div>

🌀 3. 枠線のスタイル

クラス説明
border-solid実線（デフォルト）
border-dashed破線
border-dotted点線
border-double二重線
border-none非表示
<div class="border-2 border-dashed border-red-400">破線の枠線</div>

📏 4. 角丸（rounded）

クラス説明
rounded全体を少し丸く
rounded-lgより大きく丸く
rounded-full完全な円形に
rounded-t-lg上だけ丸み
<div class="border p-4 rounded-lg">角が丸い枠線</div>

📐 5. 枠線の位置

クラス効果
border-t上だけ枠線
border-r右だけ枠線
border-b下だけ枠線
border-l左だけ枠線
border-x左右だけ
border-y上下だけ
<div class="border-x-2 border-blue-400">左右だけに枠線</div>

🌗 6. ダークモード対応

<div class="border border-gray-300 dark:border-gray-600">
  ダークモード対応の枠線
</div>

🧩 7. 応用例：フォーム

<input type="text" class="border border-gray-400 rounded-md px-4 py-2 focus:ring-2 focus:ring-blue-500" placeholder="入力してください">

🎯 まとめ

カテゴリクラス例説明
基本表示border, border-2枠線を表示、太さ変更
方向指定border-t, border-x特定方向のみ枠線
色border-blue-500色を指定
スタイルborder-dashed点線や破線など
丸みrounded-lg角を丸く
ダークモードdark:border-gray-600暗色テーマに対応
