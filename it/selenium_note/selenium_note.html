Firefoxアドオンをインストール

Firefox用Seleniumアドオンのインストール手順を以下に示す。
手順は2016年6月時点の方法である。

手順

  1. Firefoxを起動する。
  2. 公式サイトのダウンロードページにアクセスする。
  3. 「Selenium IDE」項目の「from addons.mozilla.org」をクリックし、Firefoxのアドオンサイトに遷移する。
  4. 「＋Add to Firefox」ボタンを押して、Selenium IDEアドオンをFirefoxにインストールする。
  5. Firefox再起動後、Firefoxの右上メニューに「Se」アイコンが表示されれば成功。

注意
Firefoxのアドオンサイトに直接アクセスして、 Selenium IDE をインストールするとき注意が必要である。
サイトの検索項目に「 Selenium IDE」を入力して検索すると、類似のアドオンが多数表示されるため、 どれをインストールすればいいのか迷うことになる。

Firefoxアドオンをインストール【別の方法】

注意：Firefoxのバージョンによっては、この方法でインストールできない。 SeleniumはFirefoxが使いやすい。
以下にFirefoxにSeleniumアドオンを追加する手順を示す。【2016年5月時点】

手順

  1. Firefoxを起動する。
  2. 公式サイトのダウンロードページにアクセスする。
  3. 「Selenium IDE」項目の「previous versions here」をクリックする。
  4. バージョンを選択する。（手順では 2.9.0/ を選択）
  5. xpiファイルをクリックし、道なり操作でアドオンをインストールする。
  6. Firefox再起動後、Firefoxの右上メニューに「Se」アイコンが表示されれば成功。
