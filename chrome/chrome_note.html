Chromeで画面全体のスクリーンショットを撮る

  1. スクリーンショットを撮りたいWEBページを開く
  2. Ctrl + Shift + I
  3. IPadに変更（サイズを大きくする）
  4. 右上の設定ボタン
  5. 「Chapture full size screenshot」を選択
  6. 以上で、スクリーンショット画像がダウンロードされる。
