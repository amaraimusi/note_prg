Bitbucketについて

Bitbucketとは、Gitの数あるWebサービスの１つである。
GitHubと同様なWebサービスなので、GitHubが使えれば難なく扱える

GitHubと異なり、プライベートなプロジェクトでも5人まで無料である。

bitbucketで新リポジトリを作成する

Bitbucketに新しいリポジトリを作成するには、アカウント登録が必要である。
以下の説明はすでにアカウント登録済みであることを前提にしている。

手順

  1. Bitbucketにログイン後、上段メニューのリポジトリから「リポジトリの作成」を選択する。

  2. Repository nameにプロジェクト名を入力、アクセスレベルの選択（公開、非公開の選択）、Gitを選択して作成ボタンを押すだけ。
    以上で新しいリポジトリが作成される。

Bitbucketへ別ユーザーを参加させる方法

チーム開発するため別ユーザーをリポジトリに追加することができる。
アクセス管理画面から別ユーザーを追加可能。

アクセス画面へは次の通り。
左メニュー → 設定アイコン → アクセス管理
