AWS CLIとは

  • AWS CLIはAWS Command Line Interface の略。
  • AWS関連のサービスをコマンドラインで操作するツールのこと。
  • Git Bashなどコマンドラインコンソールで使用する。
  • 無料のサービスもあれば、有料のサービスも存在する。

利用にはアクセスキー ID とシークレットアクセスキーが必要になる。

アクセスキー ID とシークレットアクセスキーの設定

まず、アクセスキーIDとシークレットアクセスキーをAWSから取得する必要がある。
AWS ユーザーガイド　アクセスキーIDとシークレットアクセスキー


	$ aws configure --profile ecr
	AWS Access Key ID [None]: アクセスキーID
	AWS Secret Access Key [None]: シークレットアクセスキー
	Default region name [None]: ap-northeast-1
	Default output format [None]: json
※2021年にリージョン(region)は大阪にできるという話がある。( 2020年の情報)

AWS CLIでECRにログイン

	$ aws ecr get-login-password --profile ecr | docker login --username AWS --password-stdin 「AWSアカウントID」.dkr.ecr.ap-northeast-1.amazonaws.com
