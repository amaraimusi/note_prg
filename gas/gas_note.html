Google Apps Scriptの特徴

Google Apps Script（以下GAS)は、ブラウザでなくサーバーで動くプログラム。
基本的な構文はJavaScriptと同じ。
GoogleドライブやスプレットシートからGASスクリプトを作成できる。
Googleドライブでスクリプトを作成すると〇〇.gsというファイルがGoogleドライブに作成される。
スプレットシートでスクリプトを作成した場合、gsファイルは作成されない。スクリプトはスプレットシートに埋め込まれている模様。
GASはスプレットシートの他にもGMAIL,カレンダーなども扱える。
Webアプリも作れるとのこと。レンタルサーバーを用意する必要がないので簡易システムを作るのに有効と思われる。
速度に難があり。GCPで高額ハイスペックエンジンを選べば早くなるかもしれない。

スプレットシートにおけるGASのHello World

スプレットシートでGASを使う時の方法。
スプレットシート以外でGASを使い場合は方法が異なるので注意

手順

  1. 新しくGASを記述するにはスクリプトエディタをまず最初に開く必要がある。
    スプレットシート上部の「ツール」→「スクリプトエディタ」でスクリプトエディタを開ける。

  2. ダイアログを表示するサンプルコードを入力。
    			function myFunction() {
			  Browser.msgBox("Hello GAS");
			}
  3. 実行ボタンを押す。最初に実行ボタンを押したときには許可が求められる。

  4. 許可は流れに沿ってボタンを押していけばよいが、「このアプリは接続されていません」というページの操作はわかりにくいので注意。 詳細リンクをクリックしてから「安全ではないページに移動」をクリックする。
  5. 再び実行ボタンを押すと、ダイアログがスプレットシート側で表示される。
    スクリプトエディタ側でダイアログが表示されるわけでないので注意すること。
  6. スプレットシート側を開くとダイアログが表示されていることが分かる。

セルの値を取得、セルに値をセット


function myFunction() {
  
  let sheet = SpreadsheetApp.getActiveSheet();
  let val = sheet.getRange(3,2).getValue(); // セルから値を取得
  console.log(val);
  
  sheet.getRange(5,6).setValue('茶色い犬'); // セルに値をセット
  
}
