html

	<div class="sec3">
		<div class="label label-default">Default</div>
		<div class="label label-primary">Primary</div>
		<div class="label label-success">Success</div>
		<div class="label label-warning">Warning</div>
		<div class="label label-danger">Danger</div>
		<div class="label label-info" style="font-size:20px">Info</div>
	</div>

説明

ボタンと記述スタイルは類似しおり、「btn」が「label」に置き換わっただけである。
ただし「btn-lg,btn-xs」などサイズを指定するセレクタはない。
代わりにfont-sizeで文字の大きさを変更すると、それに合わせてサイズも変わる。

<div>だけでなく<li>、<a>などいくつかのタグにも適用できる。
