IE11で過去バージョン5,7,8,9,10をエミュレートする

IE11で過去バージョンをエミュレートできる。
サポートしていないAPIの挙動を調べるのにとても便利。

使い方

  1. IE11を起動する。
  2. F12を押す。
  3. 「エミュレーション」タブを選択。(※下矢印によるドロップダウン表示jなっている場合もある。)
  4. ドキュメントモードからエミュレートしたいバージョンを選択する。
  5. これで選択バージョンに対応していないAPI等があればエラーになったりする。

入力フォームのいずれかにHTML5バリデーション入力エラーがあるか調べる関数

	/**
	 * 入力フォームのいずれかに１つでもHTML5バリデーション入力エラーがあればfalseを返す。
	 * pattern属性やrequired属性などのHTML5バリデーションに対応。
	 * @return true:入力エラーなし、もしくは未対応（IE9ではpatternが使えないのでtrueに）    false:いずれかに入力エラーあり
	 */
	function checkHtml5Valid(){


			var form = document.forms[0];

			for (i in form) {

				try{
					var v=form[i].validity.valid;

					if(v==false){
						return false;
					}
				}catch( e ){

				}

			}

			return true;
	}

補足文はasideタグ

補足文にはasideタグが使える。
なお、基本的にはブロック要素である。
	<aside>
	補足文章です。
	</aside>
※注意
カッコ内の補足にasideタグを使うのはふさわしくない。
また一連の文章中の中に含めるのも良くない。
あくまで独立した補足文章に用いること。
