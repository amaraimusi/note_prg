<meta>について

metaタグは、<head>内のタグで、その他の設定という意味合いがあります。
さまざまな設定ができますが、代表的なものを列挙します。

meta設定説明
<meta charset="UTF-8">文字コードの設定
<meta name="keywords" content="HTML5の覚書,沖縄,プログラマー">検索エンジン向けのキーワード指定
<meta name="description" content="検索エンジンで表示される見出し">検索エンジンで検索したときに表示される見出し
<meta name="robots" content="noindex,nofollow">検索エンジンへ、このページは検索情報に追加しないよう通知する
<meta name="viewport" content="width=device-width, initial-scale=1">スマートフォン用の初期表示領域

実体参照と文字参照

実体参照とはHTMLで「<」などの文字を「&lt;」と表記する方法である。
文字参照とは「<」を「&#60;」と数値付きで表記する。また文字参照は16進数で「&#x3C;」と表記することもできる。
特殊文字の実体参照

文字と実体参照と文字参照

文字実体参照文字参照名称
4 &#52;
C &#67;
あ &#12354;
～ &#65374;全角チルダ（波ダッシュとは別）
&&amp;&#38;アンパサンド
<&lt;&#60;小なり
>&gt;&#62;大なり

PHPにて文字を文字参照で出力するソースコード

	<?php 
	$str = "4Cあ～&<>";
	$convmap = array(0, 0x10FFFF, 0, 0x10FFFF);
	$str =  mb_encode_numericentity("$str", $convmap, 'UTF-8');
	echo htmlspecialchars($str);
	?>
出力 
	&#52;&#67;&#12354;&#65374;&#38;&#60;&#62;

OGPとは | meta og:image ...

OGPはSNSに伝える情報のことである。
対応しているSNSはFacebook,Google+,mixiなどである。
metaタグにタイトルや代表画像など、SNSに伝える情報を記述をする。

	<meta property="og:title" content="タイトル" />
	<meta property="og:site_name" content="ページタイトル" />
	<meta property="og:description" content="紹介内容など" />
	<meta property="og:type" content="website,blogなどページの種類を指定" />
	<meta property="og:url" content="ページのURL" />
	<meta property="og:image" content="ページの代表的な画像" />

フラグメント識別子とは

フラグメント識別子とはURLの「#」のことである。
「#」から右側の文字列をハッシュという。そのためハッシュ記号、ハッシュフラグメントという呼ばれ方もする。

http://example/com#hoge

accep属性について | ファイル要素で受け入れるファイルを指定する

ファイル要素で受け入れるファイルをaccept属性で指定できる。
ただし絶対的なものではないので、補助的なものであると認識すること。

acceptの指定方法はいくつか書き方があるが、ブラウザごとに挙動が異なる。 下記のサンプルは各ブラウザでなるべく共通して動くものを列挙している。

画像系 
	<input type="file" accept="image/*"/>
	<input type="file" accept="image/jpeg,image/png"/>
オーディオ系 
	<input type="file" accept="audio/*,video/*"/>

テキスト系（CSVなど） 
	<input type="file" accept="text/*"/>

ZIP 
	<input type="file" accept="application/zip"/>
