🟦 TypeScript の基本ガイド

1. TypeScript とは？

  • Microsoft が開発したオープンソースの言語。
  • JavaScript に「型」を加えたスーパーセット。
  • .ts 拡張子を使用し、JavaScript に変換して実行。

2. 型アノテーション

let message: string = "こんにちは";
let age: number = 30;
let isHappy: boolean = true;

型が違う値を代入するとエラーになります。

age = "30"; // ❌ エラー

3. 配列とオブジェクトの型

配列

let numbers: number[] = [1, 2, 3];
let names: string[] = ["Ken", "Jin"];

オブジェクト

let user: { name: string; age: number } = {
  name: "Kenji",
  age: 46
};

4. 関数の型

function greet(name: string): string {
  return "こんにちは、" + name;
}
