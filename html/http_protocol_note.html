ChromeでHTTPのヘッダ情報を確認する方法

ヘッダー情報を確認する手順
  1. 対象ページを開く
  2. F12キーを押して、デベロッパーツールを起動する。
  3. 上段メニューから Networkをクリック。

  4. F5キーを押してリロードする。HTTP通信したファイル情報が表示される。

  5. ヘッダー情報を確認したいファイルの名称をクリックする。
  6. 右側に詳細情報が表示される。Headersタブを選択するとヘッダー情報を確認することができる。
