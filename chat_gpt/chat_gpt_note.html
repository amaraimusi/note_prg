API KEYの発行方法

事前にChat GPTのアカウントを作成して置くこと。通常利用のアカウントでかまわない。
https://platform.openai.com/account/api-keysにアクセス。
「+ Create new secret key」ボタンを押し、任意の文字（英数字）を入力するとAPIキーが作成される。

APIリファレンスページからAPIキー生成ページにアクセスできる。

https://platform.openai.com/overview
上部メニューの「API reference」をクリックした先のページに、API Keysのリンクが張られているのでクリックすると、API作成ページに移動できる。
