Git Bash で composer コマンドを使う手順

✅ 環境

  • OS：Windows 10
  • ターミナル：Git Bash
  • PHP：バージョン 8.2
  • Composer：composer.phar を使用（グローバルインストールなし）

🪜 手順

① composer.phar をダウンロード

Git Bash で以下を実行して、ホームディレクトリに composer.phar を配置します。

php -r "readfile('https://getcomposer.org/installer');" | php

実行後、~/composer.phar にファイルが作成されます。

② .bashrc にエイリアスを追加

composer という短いコマンドで php composer.phar を実行できるようにします。

echo 'alias composer="php ~/composer.phar"' >> ~/.bashrc

③ .bashrc を反映させる

source ~/.bashrc

✅ これで完了！

以後、Git Bash 上で以下のように使えるようになります：

composer --version
composer install
composer update

🎯 補足：バージョン確認＆アップデート

composer self-update

で composer.phar のバージョンを最新版に更新できます。
